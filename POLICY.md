# Booza Open Collaboration Policy

Booza is built on the principles of open collaboration and transparency.

## Our Philosophy
We believe that the best software is created when everyone is free to contribute, share ideas, and learn from each other.

## Key Tenets
1. **Open Access:** All project documentation, plans, and source code are publicly accessible.
2. **Inclusive Environment:** We value diverse perspectives and encourage everyone to participate.
3. **Transparent Decision-Making:** Major project decisions are discussed and documented openly.
4. **Freedom to Modify:** Anyone is free to download, modify, and distribute the code according to the MIT License.

## How We Collaborate
We use the Conductor framework for spec-driven development, ensuring that our goals and progress are always clear.
