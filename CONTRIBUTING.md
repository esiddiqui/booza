# Contributing to Booza

Welcome! We are excited that you want to contribute to Booza, an arcade game project.

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Commit your changes with clear messages.
4. Push your branch to your fork.
5. Open a Pull Request!

## Code Style
Please follow the standard Go coding conventions (`go fmt`).

## Code of Conduct
Be respectful and kind to others in the community.
